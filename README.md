# Physics-Informed Distributed Training: a Ray / PyTorch Summit keynote

Self-contained HTML slide deck (32 slides) for the combined PC-RF + Pi-GRPO talk:
one spine (physics as a training-objective term and a validity metric), two
opposite distributed bottlenecks, and an MFU-driven bottleneck-to-primitive recipe.

Open `index.html` (or the GitHub Pages URL). Keys: arrows navigate, N toggles
presenter notes, F fullscreen, P print/export to PDF.

Author: Arun Sharma, University of Minnesota Twin Cities.
